== Description ==

New_theme (Savana) is a fully responsive Premium WooCommerce Theme with extensive functionality. Perfect for creating a website for your eCommerce business with clean and simple design.

= Features =

* Fully Responsive Design
* WooCommerce Integration
* Multiple Layout Options (1, 2, 3 columns)
* Custom Widgets
* Post Format Support (Standard, Video, Audio, Gallery, Quote, Link, Image, Aside)
* Translation Ready
* Child Theme Support
* SEO Optimized
* Cross-browser Compatible
* Custom Color Options
* Google Fonts Integration
* Mobile-First Approach
* Accessible (WCAG AA Compliant)
* GPL Licensed

= Supported Plugins =

* WooCommerce (Required)
* Contact Form 7 (Required)
* Elementor (Required)
* Yoast SEO (Recommended)
* WP Super Cache (Recommended)
* Smush (Recommended)
* MailChimp for WordPress (Recommended)
* Wordfence Security (Recommended)

== Installation ==

1. Upload the theme folder to /wp-content/themes/ directory
2. Activate the theme through the 'Appearance' > 'Themes' menu in WordPress
3. Install and activate required plugins via the notification bar
4. Go to 'Appearance' > 'Customize' to configure theme options
5. Import demo content from /demo/theme-demo-content.xml (optional)

== Frequently Asked Questions ==

= Does this theme support WooCommerce? =

Yes, New_theme has full WooCommerce support with custom styling.

= Is the theme translation ready? =

Yes, the theme includes a .pot file and supports multiple languages including English, German, Spanish, Russian, and Ukrainian.

= Can I use this theme with a child theme? =

Yes, the theme is fully compatible with child themes.

= Does the theme support all post formats? =

Yes, the theme supports all WordPress post formats: Standard, Video, Audio, Gallery, Quote, Link, Image, and Aside.
